프로젝트가 실행이 안될 때

1.이클립스의 톰캣을 중지했다가 다시 실행

2.이클립스의 톰캣을 지웠다가 다시 추가

3.이클립스 종료 후 다시 실행

4.프로젝트 선택 후 마우스 우클릭 run as > run on server로 실행

5.이클립스 종료 후 C:\Users\계정명\.m2 지우고 다시 실행 한 후 maven 업데이트

6.workspace 폴더 안의 .metadata를 지운다.
=>이클립스를 재시작후 기존 프로젝트를 import한다.

7.프로젝트 선택 후 마우스 우클릭 maven=>update project 실행

8.이클립스 파일메뉴 -> properties -> java build path에서 jre인식여부 확인

9.프로젝트 관련 폴더 중에 한글폴더가 있으면 영문폴더로 변경

10.톰캣버전을 바꿔서 설치.

11.자바버전을 다르게 설치 후 File메뉴 -> Properties -> java compiler에서 자바버전 변경
